CS 248 - Interactive Computer Graphics

CS 248 is the Stanford class that is all about the techniques behind real-time computer graphics, particularly the kind used in modern video games. For the final project in the class, Kseniya Charova and I made "Bounce!", a tile-based puzzle game (inspired by the classic Lynx/Win3.1 adventure "Chip's Challenge") rendered in stunning 3-D.
